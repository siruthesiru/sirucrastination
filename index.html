Notes

  • Definition
    • Linear Data Type
    • Data is not stored contiguously
    • Blocks of data are connected by pointers
  • The Good
    • Dynamic in size compared to Arrays
    • Inexpensive insertion and deletion
  • The Bad
    • Data has to be accessed sequentially from first node
    • Extra memory needed for the pointer

  • Linked lists are represented by nodes that point to the next
  • First node is called the head
  • The head is NULL if the list is empty

Linked List node sample definition:

	int data;
	struct node *next;
